+Sizes of grate shadows
+======================
+
+|-----------------------------------+-------+-------+-------+-------+-------|
+|                                   | Gfx16 | Gfx32 | Gfx40 | Gfx48 | Gfx64 |
+|-----------------------------------+-------+-------+-------+-------+-------|
+|-----------------------------------+-------+-------+-------+-------+-------|
+| cross flavor                      |       |       |       |       |       |
+|-----------------------------------+-------+-------+-------+-------+-------|
+| width of frame [px]               |     2 |     3 |     4 |     5 |     6 |
+| [% of tile]                       |  12.5 |  9.38 |  10.0 | 10.42 |  9.38 |
+|-----------------------------------+-------+-------+-------+-------+-------|
+|-----------------------------------+-------+-------+-------+-------+-------|
+| frame flavor                      |       |       |       |       |       |
+|-----------------------------------+-------+-------+-------+-------+-------|
+| width of frame [px]               |     3 |     6 |     7 |     9 |    12 |
+| [% of tile]                       | 18.75 | 18.75 |  17.5 | 18.75 | 18.75 |
+|-----------------------------------+-------+-------+-------+-------+-------|
+|-----------------------------------+-------+-------+-------+-------+-------|
+| Shadow of cross flavor            |       |       |       |       |       |
+|-----------------------------------+-------+-------+-------+-------+-------|
+| Frame width same as model         |       |       |       |       |       |
+|-----------------------------------+-------+-------+-------+-------+-------|
+| width of diagonals [px]           | 1+1+1 | 2+1+2 | 3+1+3 | 4+1+4 | 5+1+5 |
+| (1px strokes below+diagonal+above |       |       |       |       |       |
+| for first and second diagonal)    |       |       |       |       |       |
+|-----------------------------------+-------+-------+-------+-------+-------|
+
+Cross flavor model diagonals are scaled gfx32.
+
+Framed flavor model grid is scaled gfx32
+
+Framed flavor shadow grid is "just something that looks nice"
