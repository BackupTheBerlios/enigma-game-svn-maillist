+<h3><a class="arrowup" href="$$download$$"><img src="$$imagedir$$/nav_up.gif" border="0" title="back to top">
+  <a name="compilation">Quellcode kompilieren unter Unix</a></h3>
+
+  <p>Falls es für Ihre Unix-Distribution keine Enigma-Pakete gibt,
+  möchten Sie vielleicht die Quellpakete herunterladen und Enigma selbst
+  kompilieren. Dies ist normalerweise ziemlich einfach.</p>
+  
+  <p>Enigma hängt von einigen Bibliotheken und Werkzeugen ab. Die meisten
+  Unix-Systeme enthalten heutzutage diese Bibliotheken bereits von Anfang
+  an. Aber fürs Kompilieren benötigen Sie zusätzlich
+  '-devel'-Pakete neben den üblichen Laufzeit-Paketen. Bitte
+  stellen Sie sicher, dass die Bibliotheken SDL, SDL_ttf, SDL_image, SDL_mixer
+  und Xerces auf Ihrem System installiert sind.</p>
+  
+  <p>Für eine vollständige Liste aller Bibliotheken und Programme,
+  die fürs Kompilieren benötigt werden, lesen Sie bitte die Datei
+  'doc/REQUIREMENTS' im Enigma-Quelltext-Paket.</p>
+
+  <p>Sobald Sie alles installiert haben, geschieht das Kompilieren einfach
+  durch
+
+   <pre>./configure && make && make install</pre>
+
+  im Enigma-Verzeichnis.</p>
+  
+  <p>Falls die benötigten Pakete nicht in Ihrer Distribution enthalten
+  sind, können Sie sie auch auf den folgenden Seiten herunterladen:</p>
+
+  <ul>
+    <li><a href="http://www.libsdl.org">SDL</a> (≥ 1.2)
+    <li><a href="http://www.libsdl.org/projects/SDL_ttf">SDL_ttf</a> (≥ 2.0.6)
