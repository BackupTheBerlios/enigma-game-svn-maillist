+
+<p>Die neue stabile Version von Enigma wurde nun veröffentlicht, nach
+    einer Entwicklungszeit von mehr als 18 Monaten. Die Liste der neuen und
+    verbesserten Features umfasst:
+</p>
+
+<ul>
+    <li> Viele neue Level (jetzt mehr als 750 insgesamt)
+    <li> Unterstützung der Auflösungen 800x600 und 1024x768
+    <li> Unterstützung von nicht-englischen Übersetzungen
+    <li> Stark verbesserte Oxyd 1- und Per.Oxyd-Kompatibilität
+    <li> Neu designtes Optionsmenü
+    <li> Zahllose kleine Verbesserungen
+</ul>

+<h3 class="news">
+    <span class="date">Apr 26 2005: </span> Enigma 0.91 veröffentlicht
+</h3>
+
+<p>Wie immer tauchten ein paar unerwartete Fehler kurz nach der letzten
+    "stabilen" Version auf. Mit Enigma 0.91 wurden die folgenden
+    Probleme aus Enigma 0.90 repariert:</p>
+
+<ul>
+    <li>Falls Sie die 640x512-Auflösung in Enigma 0.81 verwendet haben,
+        betritt das Spiel nun nicht mehr in eine Endlosschleife nach dem Laden
+        der Spieldaten.
+    <li>Alle Levelpakete sind aus dem Menü heraus sichtbar, statt nur
+        "Tutorial" und "Sokoban".
+
+    <li>Plus einigen kleineren Problemen, die den meisten Nutzern sicherlich
+        nicht aufgefallen sind.
+</ul>

+    die Auswertung unserer Statistiken einzusammeln und die Weltrekordliste

+    einem Update zu 1.00 Beta wie beschrieben in <a
+    href="$$scores$$">Spielergebnisse einsenden</a> zu.
+</p>
