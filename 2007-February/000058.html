+<h3><a class="arrowup" href="$$support$$"><img src="$$imagedir$$/nav_up.gif" border="0" title="back to top">
+  <a name="forum">Forum</a></h3>
+  
+  <p>Ein Spiel wie Enigma hält einen dazu an, sich mit anderen Spielern
+  auszutauschen. Sie wollen vielleicht die Lösung eines Levels bereden
+  oder ein paar kleine Hinweise für andere suchen. Im Forum <a
+  href="http://www.mag-heut.net">mag-heut.net</a> können Sie Ihre
+  Erfahrungen mit Hunderten anderer Spieler teilen.</p>
+  
+  <p>Nur eine Bitte im Sinne aller Forumsnutzer: Bitte suchen Sie
+  zunächst nach Diskussionsfäden über dasselbe Thema,
+  z.B. denselben Level. Hängen Sie Ihre Bemerkung oder Frage via
+  "reply" an, statt einen neuen Diskussionsfaden zu beginnen.</p>
+  
+  <p>Und noch eine Bitte im Sinne unserer Levelautoren: Verderben Sie nicht
+  den Spaß an einer Level, indem Sie Komplettlösungen
+  veröffentlichen. Selbst andere Spieler bevorzugen normalerweise
+  dezentere Hinweise. Das Forum bietet dafür die Möglichkeit,
+  Hinweise mit derselben Vorder- wie Hintergrundfarbe zu schreiben (Farbe
+  "spoiler"). Leser können dann den Hinweis markieren und so
+  nach eigenem Ermessen den Hinweis lesen oder auch nicht. Halten Sie also
+  bitte die Spannung an Enigmas Rätseln für alle Spieler
+  aufrecht.</p>

+<h3><a class="arrowup" href="$$support$$"><img src="$$imagedir$$/nav_up.gif" border="0" title="back to top">
+  <a name="score">Ergebnisse und Wertungen mitteilen</a></h3>
+
+  <p>Seit Enigma 1.00 können Sie Ihre Spielergebnisse und Wertungen der
+  Levels bei uns registrieren lassen. Warum ist das sinnvoll? Auf Basis dieser
+  Daten berechnen wir die Statistiken, die Sie im Levelinspektor über
+  einen Level finden. Schließlich kann sogar das Wissen, dass viele
+  Nutzer eine bestimmte Level nicht gelöst haben, ebenso wie die besten
+  Ergebnisse zur Verbesserung von Enigma beitragen.</p>
+  
+  <p>Bitte nutzen Sie die Möglichkeit, Levels zu bewerten. Rechts- oder
+  CTRL-klicken Sie dafür auf einen Level im Levelmenü, um so den
+  Levelinspektor aufzurufen. Hier können Sie in der untersten Zeile den
+  Level mit Werten zwischen 0 (schlecht) und 10 (perfekt) bewerten. Versuchen
+  Sie dabei, sowohl gute wie schlechte Levels zu beurteilen. Durchschnittliche
+  Level sollten mit 5 bewertet werden. Ihre Wertung misst die Popularität
+  eines Levels und stellt die Grundlage dar für die monatliche Wahl zum
+  "Level des Monats".</p>
+
+  <p>Es ist nötig, dass Sie für sich einen einzigartigen
+  Spielernamen im Optionsmenü angeben. Bitte werfen Sie einen Blick auf
+  unsere <a href="$$statistics$$#userlist">Spielerliste</a>, um
+  sicherzustellen, dass Ihr favorisierter Name nicht schon vergeben ist.</p>
+
+  <p>Um dann Ihre Ergebnisse registrieren zu lassen, mailen Sie uns einfach
+  die Datei "enigma.score". Sie befindet sich in Ihrem
+  "Userpfad", der Ihnen von Enigma angezeigt wird, wenn Sie im
+  Hauptmenü F2 drücken. Bitte senden Sie uns diese einzelne Datei an  
+  <a href="mailto:score at enigma-game.org">score at enigma-game.org</a>
+  mit dem Betreff "score - Spielername" (ersetzen Sie dabei
+  "Spielername" gegen den von Ihnen gewählten Namen!).</p>
+
+  <p>Die Ergebnisse werden monatlich ausgewertet. Bitte schicken Sie sie uns
+  nicht nach jedem guten Spiel; einmal zum Ende des Monats ist völlig
+  ausreichend.</p> 
+
+  <p>Die erneuerten Statistiken werden automatisch auf Ihren Computer
+  überspielt, wenn Sie die "Update Bewertungen"-Option auf
+  "Autom." gestellt haben. Enigma wird dann ein paar Mal im Monat
+  auf dem Server nach neuen Statistiken suchen, unabhängig davon, wie oft
+  Sie die Anwendung starten. Die Netzlast ist ziemlich gering, da jeweils nur
+  Ergänzungen heruntergeladen werden, Sie erhalten so alle Updates
+  innerhalb weniger Tage.</p>
+
+  <p>Falls Sie die meiste Zeit offline spielen und die "Update
+  Bewertungen"-Option auf "Nie" gestellt haben, können Sie
+  die aktuellen Statistiken auch manuell herunterladen:
+  <a href="http://www.nongnu.org/enigma/ratings/ratings.xml.zip">ratings.xml</a>.
+  Entpacken Sie diese ZIP-Datei und kopieren Sie den Inhalt in Ihren
+  "Userpfad", in der sich schon Ihre "enigma.score"-Datei
+  befindet.</p>

+<h3><a class="arrowup" href="$$support$$"><img src="$$imagedir$$/nav_up.gif" border="0" title="back to top">
+  <a name="bug">Fehlerberichte</a></h3>
+
+  <p>Wir sind dankbar über alle Problemberichte, die Enigma
+  betreffen. Natürlich ist eine Software mit mehr als 200.000 Zeilen
+  Quelltext und komplexer Spiellogik wie Enigma niemals fehlerfrei. Aber
+  Enigma hat eine erstaunlich stabile Version 1.00 hervorgebracht, mit vielen
+  Tausend Nutzern. Bitte konsultieren Sie im Fall eines Problems zunächst
+  die Dokumentation, FAQ und das Forum für möglicherweise
+  ähnliche Fehlerberichte.</p>
+  
+  <p>Für andere Probleme als die Lösbarkeit von Levels senden Sie
+  uns bitte eine Email an die Entwickler-Mailingliste <a
+  href="mailto:enigma-devel at nongnu.org" >enigma-devel at nongnu.org</a>
+  (Sie müssen dafür nicht auf dieser Liste angemeldet sein). Ihre
+  Email sollte folgende grundlegende Informationen enthalten:
+    <ul>
+    <li>Welches Betriebssystem verwenden Sie?</li>
+    <li>Welche Version von Enigma verwenden Sie?</li>
+    <li>Was haben Sie gemacht, als der Fehler auftrat?</li>
+    <li>Was genau haben Sie dann gesehen?</li>
+    <li>Besteht das Problem weiterhin oder ist es wiederholbar?</li>
+    </ul>
+  Wenn Sie uns Ihren Fehlerbericht geschickt haben, bleiben Sie bitte am Ball
+  und beantworten weitere Nachfragen der Entwickler nach spezifischen
+  Details. Bitte erwarten Sie auch keine umgehenden Antworten - es kann
+  durchaus einige Zeit dauern, bis die richtigen Leute Ihre Frage
+  beantworten.</p>
+  
+  <p>Probleme, die sich auf einzelne Levels beziehen, sollten Sie im Forum   
+  <a href="http://www.mag-heut.net">mag-heut.net</a> posten. Bitte denken Sie
+  daran, dass alle Levels in Enigma 1.00 lösbar sind. Dennoch können
+  unbeabsichtigte Abkürzungen oder Ärgernisse in einem Level zu
+  einem Dauerthema werden und manchmal stellt sich sogar heraus, dass
+  Sie von einer Inkonsistenz in Enigma selbst verursacht werden.</p>
+  
+  <p>Entwickler können Fehlerberichte auch direkt in unserer
+  Fehlerdatenbank auf <a
+  href="http://developer.berlios.de/projects/enigma-game/">Berlios</a>
+  eintragen.
