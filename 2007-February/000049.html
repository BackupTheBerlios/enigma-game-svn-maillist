+<h3><a class="arrowup" href="$$download$$"><img src="$$imagedir$$/nav_up.gif" border="0" title="back to top">
+  <a name="compilation">Compiling from Source Code under Unix</a></h3>
+
+  <p>In case your Unix distribution does not provide Enigma packages you may
+  want to download the source package and compile Enigma for your own. This
+  is usually a simple straight forward task.</p>
+  
+  <p>Enigma depends on a few libraries and tools. Most Unix systems come with
+  these libraries pre-installed these days.  But for compilation you may need
+  additional '-devel' packages besides the runtime packages of the
+  libraries.  Please verify that the libraries SDL, SDL_ttf, SDL_image, SDL_mixer
+  and Xerces are installed on your system.</p>
+  
+  <p>For a complete list of libraries and programs that must be installed to
+  compile Enigma, please refer to the file 'doc/REQUIREMENTS' that is
+  included in the Enigma source package.</p>
+
+  <p>Once you have everything installed, building Enigma is as easy as typing
+
+   <pre>./configure && make && make install</pre>
+
+  in the enigma directory.</p>
+  
+  <p>If your distribution does not provide the necessary packages, you can
+  download them from the following web sites:</p>
+
+  <ul>
+    <li><a href="http://www.libsdl.org">SDL</a> (≥ 1.2)
+    <li><a href="http://www.libsdl.org/projects/SDL_ttf">SDL_ttf</a> (≥ 2.0.6)
+    <li><a href="http://www.libsdl.org/projects/SDL_mixer">SDL_mixer</a> 
+      (≥ 1.2.4)
+    <li><a href="http://www.libsdl.org/projects/SDL_image">SDL_image</a> 
+      (≥ 1.2.0)
+    <li><a href="http://xml.apache.org/xerces-c">Xerces</a> (≥ 2.4)
+  </ul>
