+
+

+<br>
+Participation works as follows. You need to download the levelpack
+index once from this URL "...". Copy the file to the given location, which is
+dependent on the operating system you run. For Windows it's "PATH_MS",
+for Mac OS X it's "PATH_OSX" and finally for Linux it's "PATH_UNIX".
+Then just start Enigma and enjoy the levels, each day you'll find
+one more in the "Advent 2010" levelpack.<br>
+<br>
+<br>
+We wish everybody a great deal of festive holiday fun with our Enigma Advent calendar!<br>
+<br>
+Your Enigma team
